# 3.Bonus
- 图片处理
  - pj中均使用normal文件夹中图片；
  - 在css中提前设定好各部分img的大小（像素/百分比），显示时大小便如你所愿。
- 界面美观
  - 主要界面采用了暗色系的格调（黑白配）；
  - 采用了圆角效果，使页面看起来更加柔和；
  - 页面颜色采用较少，比较简洁美观。
- 响应式设计
   - 多数元素大小通过百分比设定，浏览器宽度改变时布局收到影响较小；
   - 通过媒体查询的方法，当浏览器width小于500px时重新布局，在手机端可以正常显示，达到响应式要求。
